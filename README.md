# HOPE_INC_Customer_Management_System

Follow these steps to set up your local environment.

### 🛠️ Setup Instructions
1. **Clone the repo**: `git clone https://github.com/BaiSakinaAbad/HOPE_INC_Customer_Management_System.git`
2. **Enter directory**: `cd hopecms`
3. **Install dependencies**: `npm install`
4. **Run the project**: `npm run dev`

### Environment Check
Once the browser opens, you should see a **"WELCOME TO HOPE, INC."** screen. 
- If the background is **dark blue** and the text is **bold**, Tailwind v4 is working.
- If it looks like plain black/white text, run `npm install` again and check if `App.css` has `@import "tailwindcss";`.
